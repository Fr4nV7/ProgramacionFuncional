# ProgramacionFuncional

Este Trabajo Práctico implementa operaciones de procesamiento funcional (filtrado, reducción y agrupación) sobre cuatro dominios de datos. El módulo de Alumnos obtiene aprobados, promedios y agrupaciones por curso. El de Productos filtra por precio y calcula stocks y promedios por categoría. El de Libros filtra por páginas, cuenta por autor y encuentra el más caro. Finalmente, el de Empleados filtra por salario, calcula promedios y sumas por departamento, e identifica a los más jóvenes.
